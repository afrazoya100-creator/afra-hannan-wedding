# Hannan Shariff & Afra Zoya Wedding Invitation

## Files
- index.html
- style.css
- script.js

## How to upload to GitHub Pages
1. Create a new GitHub repository.
2. Upload these 3 files: index.html, style.css, script.js.
3. Go to Settings.
4. Go to Pages.
5. Under Branch, select main and root.
6. Click Save.
7. Wait 1–2 minutes. GitHub will show your invitation link.

## To add music
Upload an MP3 file to the repository, for example: soft.mp3
Then open index.html and change:

<source src="" type="audio/mpeg">

to:

<source src="soft.mp3" type="audio/mpeg">
